Suprapto Lost Adventure

Suprapto Lost Adventure adalah sebuah game bergenre adventure education, di dalam game ini, pemain di haruskan mencari jalan pulang menuju rumah neneknya suprapto. Namun, untuk sampai ke rumah nenek suprapto harus mencari kunci untuk membuka rumah nenek. Suprapto harus menjawab sebuah soal untuk melauinya. Pada game ini, kesulitan yang akan di hadapi suprapto adalah mencari kunci tersebut dan menjawab soal, karna soal yang akan di buat di peruntukan untuk anak usia 5 tahun ke atas, pemain akan di beri tantangan mencari sebuah kunci yang tersembunyi. Saat suprapto mencari kunci, suprapto akan di berikan pertanyaan dan dia harus menjawabnya untuk mendapatkan kunci

Game Overview

Apa saja yang ada di dalam game?

Metode Pembuatan

Suprapto Lost Adventure di buat dengan game maker Unity dan Blender sebagai tools pembuatan asset object 3D. game ini dibuat dengan cinta dan kasih sayang disetiap ketikan keyboard dan pergerakan mouse

Control

Game dengan kontrol yang mudah, pemain hanya menggerakan arahan mata dengan mouse dan arahan kaki dengan "W" sebagai bergerak maju "A" untuk bergerak ke kiri "S" untuk mundur, dan "D" untuk ke kanan.

Publikasi

Semua orang bisa memainkan game ini, untuk link download sudah di sertakan pada page paling atas atau HOME

online?

pada saat sekarang game ini offline, karena jalan game ini bedasarkan cerita dan player bisa menikmati sebuah map yang luas dan harus mencari kunci untuk menyelesaikannnya.

Platform

Suprapto Lost Adventure akan rilis di platform PC karena grafis visual dan audio akan di fokuskan dalam game ini

Challanges

Player akan di hadapkan pada pertanyaan setiap ingin mengambil kunci. di setiap pertanyaan player harus memilih satu di antara dua opsi sebagai jawabannya.
